# 기능 구현 목록

## 체스판
- [ ] 체스판을 생성한다.

## 체스말
- [ ] 자신의 최초 위치를 알려준다.

## 입력
- [ ] 사용자는 start를 입력해 게임을 시작할 수 있다.
- [ ] 사용자는 end를 입력해 게임을 종료할 수 있다.
  - [ ] start / end가 아닌 입력은 예외가 발생한다.

## 출력
- [ ] 만들어진 체스판을 출력한다.
